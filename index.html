Generate Synthetic Datasets

This script will create a synthetic dataset with a structure similar to the one you would get from user interactions. The ratings array is filled with random integers between 1 and 5, representing the ratings given by users to movies. The mask array is used to simulate the fact that not all users have rated all movies. The unrated movies are represented by zeros in the ratings array.

The resulting synthetic_ratings.csv file can then be used to train your collaborative filtering model. Remember that synthetic data can be useful for testing and development purposes, but it may not capture the complexity and nuances of real-world user behavior.
